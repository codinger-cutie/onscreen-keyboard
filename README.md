# onscreen-keyboard

Inspired by [Kashumy's similar project](https://github.com/Kashumy/itxtedit.github.io)

### Features

- [ ] Code/text editor
- [ ] Onscreen keyboard that can be hidden
- [ ] Keyboard has navigation for mobile devices
- [ ] Text selection
- [ ] Saving text into localStorage
- [ ] Responsive
- [ ] Color themes
